# ✈️ FOD Safety Analytics Dashboard

## 📌 Project Description
Built an end-to-end FOD Safety Analytics Dashboard using Python, Excel, and Tableau to analyze 50K+ incident records. Performed data cleaning, EDA, and visualization to identify high-risk zones, peak incident periods, and major contributing factors through KPI dashboards, heatmaps, and trend analysis.

---

## 🔍 Problem Statement
Foreign Object Debris (FOD) incidents in airfield operations can lead to serious safety risks, operational delays, and equipment damage. Identifying patterns and understanding root causes from large datasets is difficult without proper analytics and visualization.

---

## 💡 Solution
Developed an interactive analytics dashboard that helps monitor FOD incidents, analyze trends, and uncover high-risk patterns using data-driven insights.

---

## ⚙️ Tech Stack
- Python (Pandas, NumPy)
- Tableau
- Excel
